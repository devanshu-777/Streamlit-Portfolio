# Streamlit Portfolio

A modern, interactive portfolio web app built with [Streamlit](https://streamlit.io/).  
Showcase your skills, projects, and resume with a clean, responsive design.

---

## 🚀 Project Description

This project is a personal portfolio web application developed using Streamlit.  
It highlights your professional background, technical skills, and key projects, and provides quick access to your resumes.  
The app is easy to customize and deploy, making it ideal for students, developers, and professionals who want a web-based portfolio with minimal setup.

---

## 🛠️ Setup Instructions

### 1. Clone the repository

```sh
git clone https://github.com/devanshu-777/Streamlit-Portfolio.git
cd REPO-NAME
```

### 2. Create and activate a virtual environment (recommended)

```sh
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```sh
pip install -r requirements.txt
```
*(If you don’t have a `requirements.txt`, create one with: `pip freeze > requirements.txt` after installing your packages.)*

### 4. Run the Streamlit app

```sh
streamlit run portfolio_app.py
```

---

## ⚙️ Useful Commands

- **Install a new package:**  
  `pip install package-name`
- **Freeze dependencies:**  
  `pip freeze > requirements.txt`
- **Deactivate virtual environment:**  
  `deactivate`
- **Stop the app:**  
  Press `Ctrl+C` in the terminal.

---

## 📝 Customization

- **Profile Picture:**  
  Replace `profile.jpeg`, `profile.jpg`, or `profile.png` in the project folder.
- **Resume Links:**  
  Update the Dropbox/Google Drive links in `portfolio_app.py`.
- **Project Info:**  
  Edit the project sections in `portfolio_app.py` to add your own projects and GitHub links.
- **Contact Form:**  
  Replace `YOUR_EMAIL_HERE` in the contact form with your email address.
